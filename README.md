# Portfolio Revision

Projet de révision pour pratiquer HTML, CSS et JavaScript.  
Ce projet permet de revoir les bases tout en ajoutant des fonctionnalités modernes comme le dark mode et la validation de formulaire.

---

## Aperçu

![Capture d'écran](images/screenshot.jpg)

Le projet est responsive et fonctionne sur mobile et desktop.

---

## Fonctionnalités

- Formulaire de contact avec validation et messages dynamiques
- Dark mode avec toggle (soleil / lune)
- Cartes de projets responsives
- Transitions CSS subtiles pour les liens et boutons
- Accessible et responsive

---

## Technologies utilisées

- HTML5
- CSS3 (Flexbox, variables, transitions)
- JavaScript (DOM, événements, validation formulaire)
- Git / GitHub pour le contrôle de version
